# Инструкция по ручной загрузке галереи работ

## 📋 Что нужно сделать

Для того чтобы лендинг выглядел профессионально, нужно добавить реальные примеры работ в галерею.

## 📁 Структура файлов

Галерея находится в папке:
```
/opt/tattoo-bot/landing/gallery/
```

## 🖼️ Требования к изображениям

### Форматы:
- **JPG** (рекомендуется) — качество 85%, progressive
- **PNG** — для изображений с прозрачностью
- **WebP** — оптимально для веба

### Размеры:
- **Основное изображение:** 800×800 пикселей (квадрат)
- **Миниатюра:** 400×400 пикселей
- **Соотношение:** 1:1 (квадрат)

### Названия файлов:
```
tattoo-001.jpg
tattoo-002.jpg
tattoo-003.jpg
...
```

## 📝 Информация к каждому изображению

Для каждого эскиза нужно создать файл с описанием в формате JSON:

**Пример:** `tattoo-001.json`
```json
{
  "id": "001",
  "title": "Дракон на спине",
  "description": "Трайбл дракон с геометрическими элементами",
  "style": "Трайбл/Геометрия",
  "placement": "Спина",
  "generation_time": "45 сек",
  "date": "2024-04-15",
  "keywords": ["дракон", "трайбл", "геометрия", "спина"],
  "visible": true
}
```

## 🔧 Интеграция с HTML

Шаблон для отображения в галерее:

```html
<div class="gallery-item" data-id="001">
    <div class="gallery-image">
        <img src="gallery/tattoo-001.jpg" alt="Дракон на спине" loading="lazy">
    </div>
    <div class="gallery-info">
        <h3>Дракон на спине</h3>
        <p><strong>Стиль:</strong> Трайбл/Геометрия<br>
           <strong>Расположение:</strong> Спина<br>
           <strong>Время генерации:</strong> 45 сек</p>
    </div>
</div>
```

## 🚀 Процесс добавления работ

### Шаг 1: Подготовка изображений
1. Выбрать лучшие работы из бота
2. Обрезать до квадрата 800×800px
3. Оптимизировать (меньше 200KB на файл)
4. Сохранить в папку `gallery/`

### Шаг 2: Создание описаний
1. Для каждого изображения создать `.json` файл
2. Заполнить всю информацию
3. Указать `"visible": true` для показа

### Шаг 3: Обновление HTML (автоматически)
Скрипт будет автоматически генерировать галерею из JSON файлов.

**Файл для автоматической генерации:** `js/gallery-loader.js`

## 🛠️ Инструменты для обработки изображений

### Онлайн:
- **TinyPNG** — сжатие без потери качества
- **Squoosh** — оптимизация от Google
- **Canva** — обрезка и базовое редактирование

### Локально (если установлены):
```bash
# Установка ImageMagick (если нет)
apt-get install imagemagick

# Обрезка до квадрата
convert input.jpg -resize 800x800^ -gravity center -extent 800x800 output.jpg

# Оптимизация
convert output.jpg -strip -quality 85 -interlace Plane optimized.jpg
```

## 📊 Пример хороших работ для галереи

### Категории:
1. **Животные** — драконы, волки, львы, змеи
2. **Растения** — розы, лотосы, папоротники
3. **Надписи** — цитаты, имена, даты
4. **Символы** — мандалы, геометрические узоры
5. **Портреты** — лица, профили, силуэты

### Стили:
- Реализм
- Минимализм
- Акварель
- Трайбл
- Олд скул / Нью скул
- Дотворк
- Геометрия

## ⚠️ Важные моменты

### Что НЕ размещать:
- Эскизы низкого качества
- Нечёткие/размытые изображения
- Изображения с водяными знаками конкурентов
- Контент 18+ (без цензуры)
- Работы без разрешения авторов

### Авторские права:
- Использовать только оригинальные работы из бота
- Если используются референсы — иметь разрешение
- Указывать источник вдохновения при необходимости

## 🔄 Обновление галереи

### Периодичность:
- **Еженедельно** — добавлять 2-3 новые работы
- **Ежемесячно** — удалять старые/менее популярные работы
- **По запросу** — добавлять работы пользователей (с их разрешения)

### Процесс:
1. Добавить новые файлы в папку `gallery/`
2. Обновить JSON описания
3. Проверить отображение на сайте
4. Протестировать на мобильных устройствах

## 📞 Поддержка

При возникновении вопросов:
1. Проверь эту инструкцию
2. Посмотри примеры в папке `gallery/examples/`
3. Обратись к разработчику лендинга
