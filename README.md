# Ink Mirror Landing Page

Лендинг для AI-генератора эскизов тату.

## 🎯 О проекте

Ink Mirror — это AI-генератор, который превращает идеи в профессиональные эскизы тату за 1 минуту. Лендинг служит центральным хабом для всех платформ и направляет трафик в Telegram бота (@mynewtattoobot).

## 🏗️ Структура

```
/opt/tattoo-bot/landing/
├── index.html              # Главная страница
├── css/
│   └── styles.css         # Стили (адаптивный дизайн)
├── js/
│   └── script.js          # Интерактивность
├── images/

│   └── favicon.svg        # Иконка сайта
└── gallery/               # Галерея работ (будет заполнена вручную)
```

## 🚀 Развёртывание на GitHub Pages

### Вариант 1: Через ветку `gh-pages`
```bash
cd /opt/tattoo-bot/landing
git init
git add .
git commit -m "Initial landing page"
git branch -M main
git remote add origin https://github.com/username/inkmirror-landing.git
git push -u origin main

# Создать ветку gh-pages
git checkout -b gh-pages
git push -u origin gh-pages
```

### Вариант 2: Через папку `docs`
1. Переименовать папку `landing` в `docs`
2. Включить GitHub Pages в настройках репозитория (Source: docs folder)

### Вариант 3: Через Vercel/Netlify (рекомендуется)
- Просто загрузить папку на Vercel или Netlify
- Автоматический деплой, SSL, CDN
